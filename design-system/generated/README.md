# RECURSOS GENERADOS

Esta carpeta contiene recursos creados a partir de contratos visuales aprobados.

Cada recurso debe conservar un archivo de metadatos con:

- `piece_id`;
- ruta del contrato visual;
- modelo y snapshot;
- hash del prompt;
- referencias utilizadas;
- fecha de generación;
- modo de ejecución;
- ruta del preview aprobado cuando corresponda.

Los assets finales no se regeneran desde cero si ya existe una versión aprobada que debe conservarse.
