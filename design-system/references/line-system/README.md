# REFERENCIAS — LINE SYSTEM

Familia basada en diagramas simples de líneas, nodos, trayectorias y relaciones.

## Rasgos aprobados

- una relación visual dominante;
- geometría limpia y precisa;
- líneas negras o de la paleta Motion;
- nodos circulares usados con función semántica;
- fondos planos;
- mucho espacio negativo;
- pocos elementos;
- lectura inmediata en móvil;
- calidad cercana a una ilustración editorial, no a un wireframe improvisado.

## Referencias conceptuales

- estilo Matt Gray adaptado a Motion;
- esquema comparativo de Mejora Continua, Gestión del Cambio y Transformación Continua compartido por Paulo;
- ejemplos aprobados donde una línea conecta, se enreda, se separa, converge o cambia de dimensión.

## No usar

- sombras volumétricas;
- cintas 3D;
- objetos escultóricos;
- texturas materiales;
- gradientes realistas;
- redes de nodos decorativas;
- mezcla con arte conceptual volumétrico.

Las referencias visuales concretas deben almacenarse en esta carpeta y pertenecer únicamente a esta familia.
