# REFERENCIA VISUAL — CONCEPTO 2

**Fuente:** archivo entregado por Paulo el 18 de julio de 2026.  
**Rol:** fuente principal de inspiración visual.  
**Uso:** referencia conceptual interna; no usar imágenes de terceros como assets finales sin licencia.

## Qué aporta

La colección amplía el repertorio de Motion para explicar conceptos complejos mediante infografías editoriales minimalistas.

Patrones observados:

- geometrías dominantes de gran escala;
- círculos, semicírculos, bloques, cintas, espirales y escalones;
- escala y proporción utilizadas como significado;
- espacio negativo activo;
- formas que cambian de tamaño, densidad, continuidad o posición;
- timelines, rutas y progresiones;
- superposiciones, transparencias e intersecciones;
- volúmenes simples y pliegues;
- siluetas humanas pequeñas frente a estructuras conceptuales;
- datos convertidos en una interpretación visual;
- bocetos y pensamiento gráfico;
- paleta contenida y tipografía subordinada a la idea.

## Traducción a Motion

No se copia la paleta roja ni las composiciones de stock.

Se traduce el principio a:

- colores oficiales Motion;
- Futura, Lyon y Gotham según su función;
- formas plegadas, ecos, estrías y deformaciones del branding;
- composición limpia;
- una idea visual dominante;
- variedad de recursos;
- ejecución determinística, generativa o híbrida según el concepto.
