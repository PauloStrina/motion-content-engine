# REFERENCIAS — CONCEPTUAL ART

Familia basada en una metáfora visual dominante con profundidad, volumen, materia, sombra o composición artística.

## Rasgos aprobados

- una sola metáfora central;
- profundidad y sombra controladas;
- pliegues, cintas, capas u objetos con intención;
- composición editorial limpia;
- alto nivel de terminación visual;
- paleta Motion integrada en el objeto, no aplicada como decoración;
- tipografía y logo compuestos posteriormente con precisión;
- capacidad de condensar una idea compleja en una imagen memorable.

## Referencias conceptuales

- composiciones generadas y aprobadas por Paulo para mostrar integración entre Negocio, Tecnología y Cultura;
- cintas plegadas y objetos escultóricos con sombra;
- escenas conceptuales donde la forma representa una tensión o transformación.

## No usar

- redes de puntos como recurso principal;
- curvas y nodos tipo infografía;
- múltiples minidiagramas;
- apariencia de dashboard;
- texto o logo generado dentro de la imagen;
- mezcla con el sistema Matt Gray dentro del mismo asset.

Las referencias visuales concretas deben almacenarse en esta carpeta y pertenecer únicamente a esta familia.
