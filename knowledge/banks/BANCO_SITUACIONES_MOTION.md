## Reglas de uso

1. El Banco de Situaciones no prueba que exista un caso real.
2. Para contenidos de Resultados, toda situación debe enlazarse con una evidencia autorizada.
3. Para Problema o Método puede utilizarse una situación observable sin atribuir cliente, cifra o resultado.
4. Cada nueva situación recibe un ID estable y se conecta con uno o más artefactos.
5. Las correcciones se actualizan en ambos repositorios mediante PRs equivalentes.
