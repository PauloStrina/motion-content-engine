# Motion graphics programáticos (tipografía cinética)
Método: las escenas del guion (tabla del Redactor) se convierten en una página HTML animada con los tokens Motion
(animaciones CSS por escena), y se renderiza a video capturando frames con Puppeteer + ensamblando con ffmpeg
(o usando Remotion si el equipo prefiere React). Plantilla inicial: TODO en F2.b — primer video se produce manualmente
guiado para fijar el estándar visual, después se templatiza.
